# BX_TiKuBaseModulesRepo
题库基础库，支持富文本,支持多种题型

用法：
 Podfile 中添加 
 `source 'https://github.com/CocoaPods/Specs.git'  # 官方库`
 `source 'https://github.com/QinKingHan/BX_Podspec_Repo.git'   # 私有库`
 
 `pod 'TiKuBaseModulesRepo', '~> 0.3.0'`
  

最后 pod install
